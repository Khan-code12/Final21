
Electricity Billing System (PHP + MySQL)

Setup (XAMPP/WAMP)
1) Copy 'electricity-billing' folder to your web root (htdocs).
2) Start Apache & MySQL.
3) Import sql/schema.sql in phpMyAdmin.
4) Open http://localhost/electricity-billing/index.php
